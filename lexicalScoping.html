Lexical Scoping

  • Lexial scope is a Rule, which is decided when you write a code.
  • Lexical scope is where you declare variables and functions, not where you called them.
    There are 4 type of scopes in JavaScript
  1. Global Scope: Access anywhere in the code
  2. Function Scope: Access only inside the function
  3. Block Scope: Variables declare using 'let' or 'const' inside conditional statements or control structure
  4. Module Scope: Access only inside that module unless it is exported
